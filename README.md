* [Server Space](https://serverspace.io) → FreeBSD 14 (verified on 31 March 2026)
* [Vultr](https://www.vultr.com) → FreeBSD 15, 14, and 13, as well as OpenBSD 7.8 and 7.7 (verified on 31 March 2026)
