## Cloud Providers
------------------

The following cloud providers have been verified to support BSD operating systems:

| Name | URL | Notes | Verified Date |
| --- | --- | --- | --- |
| AWS Lightsail | https://aws.amazon.com/lightsail | Freebsd 15.0, 14.3, and 13.5 | 31 March 2026 |
| Hawk Host | https://www.hawkhost.com | Freebsd 14 and 13 | 31 March 2026 |
| Server Space | https://serverspace.io | Freebsd 14 | 31 March 2026 |
| Vultr | https://www.vultr.com | Freebsd 15, 14, and 13, as well as Openbsd 7.8 and 7.7 | 31 March 2026 |

Note: The verified dates refer to the date of verification for each provider.
